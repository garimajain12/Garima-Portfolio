## Sections

- **Home** - Hero section with animated introduction
- **About** - Personal information and statistics
- **Work** - Portfolio projects showcase with live demos
- **Experience** - Professional timeline with collapsible details
- **Skills** - Technical skills organized by categories with brand logos
- **Contact** - Contact form and social links

## Getting Started

### Installation

1. Install dependencies:
```bash
npm install
```

### Development

Run the development server:
```bash
npm run dev
```

The site will be available at `http://localhost:5173`

## Customization

1. **Update Personal Information**: Edit the content in each component file:
   - `src/components/Home.jsx` - Update name and title
   - `src/components/About.jsx` - Update about text and stats
   - `src/components/Work.jsx` - Update projects array
   - `src/components/Experience.jsx` - Update experience array
   - `src/components/Skills.jsx` - Update skills array
   - `src/components/Contact.jsx` - Update contact information

2. **Theme**: The theme system is managed in `src/context/ThemeContext.jsx`
